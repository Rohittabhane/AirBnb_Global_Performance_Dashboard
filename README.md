# Airbnb Global Performance Dashboard

This project analyzes Airbnb listing performance across multiple cities, focusing on pricing trends, user ratings, and seasonal demand patterns.

The objective was to simulate a real-world business intelligence scenario by transforming raw data into actionable insights using Power BI.

## Key Insights

- Identified seasonal demand variations across cities using ribbon charts
- High review frequency correlates with better-rated listings, indicating engagement impacts perception
- Pricing patterns vary significantly by city, highlighting location-based strategy importance

### Behavioral Observations
- Peak booking activity aligns with seasonal travel trends
- Certain cities maintain consistent demand, while others are highly seasonal

## Technical Implementation

- Cleaned and transformed raw dataset using Power Query
- Built data model with relationships between listings, reviews, and calendar data
- Created DAX measures for:
  - Performance KPIs
  - Rating analysis
  - Seasonal comparisons
- Implemented bookmarks for dynamic view switching (Overview vs Detailed insights)

## Business Impact

This dashboard demonstrates how data can support:

- Pricing strategy optimization across locations
- Identifying high-performing listings
- Understanding seasonal demand fluctuations
- Enhancing customer experience through review insights
